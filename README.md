# Productivity-App-Conquer

A React + Vite productivity dashboard built with components for tasks, notes, calendars, goals, and more.

## Local development

```bash
git clone https://github.com/coyote-07/Productivity-App-Conquer.git
cd conquer
npm install
npm run dev
```

## Features

- React + Vite app shell
- Dashboard, tasks, notes, calendar, goals, and templates pages
- Sidebar and workspace context
- Custom UI widgets and components
